Module With_loc

When the symbol you want to decorate is not already an argument in a record, it may be convenient to use this type as a standard way to decorate a symbol with a position.

Using Loc.t or 'a With_loc.t are two different styles to decorate a symbol, you may choose one depending on the context.

An example using Loc.t:

type t =
  | T of { loc : Loc.t ; a : A.t; b : B.t; ... }

An example using 'a With_loc.t:

type t = A.t With_loc.t list
type 'a t = {
  1. loc : Loc.t;
  2. symbol : 'a;
}
